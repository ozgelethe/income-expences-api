## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
* pip
```sh
download python
```


### Installation

1. Clone the repo
```sh
git clone https://github.com/rukiyearican/income-expences-api.git
```
2. Install requirements.txt
```sh
pip install requirements.txt
```
